# Gestao-de-Estoque-
Projeto desenvolvido em Python para gerenciamento simples de estoque via terminal. O sistema permite cadastrar produtos, registrar entradas e saídas de mercadorias, consultar quantidades disponíveis e gerar relatórios completos do estoque.

Funcionalidades
Cadastro de produtos com:

. ID automático

. Nome

. Categoria

. Preço

. Quantidade

. Registro de entrada de produtos

. Registro de saída de produtos

. Consulta de estoque por ID

. Relatório completo dos produtos cadastrados

. Alerta de estoque baixo

. Menu interativo no terminal
